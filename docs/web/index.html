/plugin install vocli@vocli
# Setup voice
/vocli:install

Why voCLI?

🔐

100% Private

All audio processing happens on your machine. Nothing is sent to the cloud. Ever.

🎤

Natural Voice

Powered by Kokoro TTS with 54 voices. Switch between American, British, male, female — your choice.

⚡

Fast Recognition

faster-whisper STT with auto-detection of Apple Silicon vs Intel for optimal performance.

🌐

Remote Ready

Claude Code on a VM? Run audio servers locally, connect over SSH. Voice works anywhere.

🔄

Engine Choice

Kokoro for quality, Piper for speed, macOS Say as fallback. Switch anytime via config.

💬

Conversational

Voice stays active during your entire session. Drop files, paste text — it keeps talking.
